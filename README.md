# 💱 Currency Converter

A simple Java-based Currency Converter that allows users to convert an amount from one currency to another using a menu-driven interface. The project focuses on basic Java concepts and accurate currency calculations.

## ✨ Features

- Convert between multiple currencies
- Menu-driven interface
- Input validation using try-catch
- Handles invalid and negative inputs
- Accurate calculations using BigDecimal
- Multiple conversions in a single run

## 🛠 Technologies Used

- Java
- VS Code
- JDK

## 📚 Concepts Used

- Scanner
- Methods
- switch-case
- do-while loop
- try-catch
- BigDecimal

## 🎯 Learning

This project helped me improve my understanding of Java programming, input validation, methods, loops, exception handling, and currency conversion logic.
